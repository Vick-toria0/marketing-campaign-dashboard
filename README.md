# Marketing Campaign Performance Dashboard

# Overview
Analysis of 500 marketing campaigns to uncover performance trends across channels, countries, and campaign types. Built during my Data Analytics internship at NextMakers, this project transforms raw, unstructured campaign data into a clear, interactive dashboard that supports data-driven marketing decisions.

# Objective
To clean, analyze, and visualize marketing campaign data in order to identify which channels, countries, and strategies actually drive conversions — and to challenge assumptions about what makes a campaign successful.

# Tools Used
- Microsoft Excel — data cleaning, VLOOKUP, pivot tables, conditional formatting
- Power BI — interactive dashboard design and data visualization

# Process
1. Cleaned and prepared a raw dataset of 500 marketing campaigns in Excel
2. Built pivot tables to summarize performance by channel, country, and campaign type
3. Ran correlation analysis between ad spend and conversions
4. Designed an interactive Power BI dashboard to visualize key metrics and trends


# Key Insights

Instagram is the strongest channel: Instagram generated the highest conversions (22,625) from just 110 campaigns, outperforming Facebook Ads, Google Ads, Referral, and Email.

The USA converts more efficiently than Kenya: The USA had the highest conversions overall (18,201), despite Kenya spending slightly more on ads ($92,861 vs. $92,455). This shows the USA converted more efficiently per dollar spent.

Clicks don't equal conversions: Product Launch campaigns generated far more clicks (353,412) than Holiday Promo campaigns, yet Holiday Promo converted better — proving that engagement alone doesn't predict results.

Budget size barely matters: Correlation analysis between ad spend and conversions returned a near-zero score (0.04) — meaning bigger budgets don't reliably produce better outcomes.

Strategy beats spending: 111 campaigns had high budgets but were still classified as Low Performers, while 20 Low Budget campaigns became Top Performers. This was the most significant finding of the project: smart targeting and strategy consistently outperform raw ad spend.
